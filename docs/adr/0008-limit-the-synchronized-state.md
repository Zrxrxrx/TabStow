# Limit the Synchronized State

Automatic Gist synchronization will include Saved for Later, Quick Links, and independently versioned `includePinnedTabs` and `closePinnedTabs` behavior preferences. OAuth credentials, Gist ID and filename, the locally owned Replica ID, History, Todos, theme, and language remain Device-local State; the opaque Replica ID may appear inside revisions but is never imported as a setting or replaced remotely.
