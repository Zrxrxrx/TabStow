# Gate Automatic Sleep by Capability

Tabstow will preserve extension availability on Chrome versions below 121 rather than declare a hard `minimum_chrome_version`. Automatic sleep alone will be unavailable when the runtime does not expose a valid `Tab.lastAccessed`, and its settings control will explain that Chrome 121 or later is required; manual sleep, Sleeping state, Observed Sleep Periods, and Stow Suggestions remain available. Automatic sleep will never substitute a tab's opening time or Tabstow's first observation time because those values do not mean Tab Inactivity. Runtime scans continue to fail closed when `lastAccessed` is missing, invalid, or in the future even on a nominally supported browser.
