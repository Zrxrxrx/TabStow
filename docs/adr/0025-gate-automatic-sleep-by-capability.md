# Gate Automatic Sleep by Capability

Tabstow requires Chrome 114 or later because its toolbar surface uses Chrome's native Side Panel API. Within that supported range, Chrome 114–120 retains every feature except Automatic sleep rather than raising the minimum to Chrome 121. Automatic sleep remains unavailable when the runtime does not expose a valid `Tab.lastAccessed`, and its settings control explains that Chrome 121 or later is required; manual sleep, Sleeping state, Observed Sleep Periods, and Stow Suggestions remain available. Automatic sleep will never substitute a tab's opening time or Tabstow's first observation time because those values do not mean Tab Inactivity. Runtime scans continue to fail closed when `lastAccessed` is missing, invalid, or in the future even on a nominally supported browser.
