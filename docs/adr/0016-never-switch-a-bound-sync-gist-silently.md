# Never switch a bound Sync Gist silently

Once a device establishes a Gist Binding, automatic synchronization will use only that exact Gist ID and Sync File name. A missing Gist, renamed file, or lost access enters Synchronization Paused while retaining local data and Pending Synchronization; rediscovery and rebinding occur only during initial connection or after an explicit user action in Settings, never as an automatic fallback.
